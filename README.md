This project performs an **Exploratory Data Analysis (EDA)** on the Netflix dataset to understand the content strategy of the platform. Using Python libraries like **Pandas, Plotly, and Seaborn**, we analyze trends over time, content distribution (Movies vs. TV Shows), and production dominance by country.

## ğŸ“Š Key Insights & Visualizations
The analysis covers the following aspects:

1.  **Content Distribution:** Comparison between the number of Movies and TV Shows.
2.  **Growth Trend:** How Netflix content has grown over the years.
3.  **Duration Analysis:** Histogram of movie durations and TV show seasons.
4.  **Global Reach:** Top 10 content-producing countries.
5.  **Word Cloud:** Most frequent words used in titles.
